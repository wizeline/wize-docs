# First non repeated letter

Write a method that for a given string as input, returns the first non repeated letter.

## Interviewer Statement
```
// Write a method that for a given string as input, returns the first non repeated letter.
```

## Test cases
```
'abcdefghija' => b
'hello' => h
'Java' => J
'simplest' => i
```

## Solutions
- Using data structures, a hashmap where you store each letter as key.
